# Front-matter

## 主题特有

| 参数          | 描述                     | 默认值                 |
| :------------ | :----------------------- | :--------------------- |
| `cover`       | 封面，需为图片目录或链接 | config.background.path |
| `coverWidth`  | 封面宽度                 | 1600                   |
| `coverHeight` | 封面宽度                 | 900                    |
| `reprint`     | 是否为转载               | false                  |
